# spec-kit 예제 (Youtube Downloader)

## speckit-constitution

```sh
/speckit-constitution

한글로 작성해줘. 유튜브에서 원하는 동영상을 다운로드하는 프로그램을 개발할거야. 파이썬으로 백엔드를 만들고, Vue3로 프론트엔드를 만들거야. 많이 사용하는 유명한 파이썬 라이브러리로 유튜브 영상을 다운로드할거야. 앞으로 개발은 TDD의 Red Green으로 진행해줘. 명확하게 정의한 다음 코드리뷰를 해서 공통 로직이 있는지 리팩토링할 것이 있는지 검사해서 마무리해줘.
```

## speckit-specify

```sh
/speckit-specify

## 기능요구사항
- GUI 화면에 유튜브 URL 입력 기능이 있음
- 입력받은 URL을 이용해서 유튜브 다운로드 기능 실행
- 저장옵션 선택(ex. mp4, mp3)
- 다운로드 진행상황 표시

## 기술스택
- Flutter
- Android SDK
```
